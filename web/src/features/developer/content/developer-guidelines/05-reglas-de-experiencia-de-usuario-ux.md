# ✅ Reglas de experiencia de usuario (UX)

El diseño de los juegos debe cumplir **criterios básicos de usabilidad**, garantizando una experiencia clara, accesible y fluida para el usuario.

> [!IMPORTANT]
> La experiencia de usuario debe facilitar que el jugador entienda qué puede hacer, qué está ocurriendo y cuál es el estado de la partida en cada momento.

## Inicio rápido

El usuario debe poder iniciar una partida en **pocos pasos**, evitando fricción innecesaria.

## Interfaz clara

La interfaz debe ser **simple, intuitiva y sin elementos innecesarios**, priorizando la información relevante.

## Feedback inmediato

El sistema debe informar al usuario de forma clara sobre:

* Acciones realizadas.
* Turnos.
* Resultados.

## Visibilidad del estado

El jugador debe conocer en todo momento:

| Elemento                 | Descripción                                         |
| ------------------------ | --------------------------------------------------- |
| **Estado de la partida** | Situación general de la partida.                    |
| **Progreso**             | Avance del jugador o de la partida.                 |
| **Situación actual**     | Contexto inmediato necesario para tomar decisiones. |

## Consistencia

Los elementos de interfaz y comportamiento deben mantenerse **coherentes a lo largo del juego** para evitar confusión.
